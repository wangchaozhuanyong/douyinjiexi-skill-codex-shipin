# Content Formats

Use one format per video. A format is the structural promise, not a decoration.

## mistake_correction

Correct a common wrong method.

- Hook: name the mistake quickly.
- Body: show wrong input/output, then corrected input/output.
- Save value: reusable correction rule.

## three_step_tutorial

Teach one workflow in three steps.

- Hook: show the result or pain.
- Body: step 1, step 2, step 3 with proof for each step.
- Save value: the three-step checklist.

## before_after

Compare a weak method and a stronger method.

- Hook: put the weak result beside the strong result.
- Body: explain what changed.
- Save value: the change list.

## news_explain

Explain an AI update with sources.

- Hook: why this update matters now.
- Body: what changed, who is affected, what to try.
- Save value: a practical decision rule.
- Required: source title, URL or local note, and date.

## tool_stack

Combine tools into one practical workflow.

- Hook: the final result.
- Body: tool A -> tool B -> tool C.
- Save value: stack diagram and usage order.

## prompt_template

Teach a reusable prompt.

- Hook: show the bad prompt result first.
- Body: build the template piece by piece.
- Save value: full copyable template.

## case_breakdown

Break down a real or concept case.

- Hook: show the finished artifact or result.
- Body: how it was made, what worked, what failed.
- Save value: reusable lessons.

## myth_busting

Break a common misconception.

- Hook: state the misconception and correction.
- Body: show evidence that makes the correction believable.
- Save value: judgment standard.
