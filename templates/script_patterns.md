# Script Patterns

## 1. 错误纠正型

开头指出错误 -> 展示错误结果 -> 给出正确结构 -> 展示修正结果 -> 给可复用模板。

## 2. 三步教程型

先展示结果 -> 步骤一 -> 步骤二 -> 步骤三 -> 回到结果和保存清单。

## 3. 前后对比型

左边普通做法 -> 右边改进做法 -> 拆解差异 -> 给判断标准。

## 4. 新闻解释型

一句话说明变化 -> 引用来源和日期 -> 解释影响对象 -> 给出可尝试动作。

## 5. 工具组合型

展示最终产物 -> 工具 A 负责什么 -> 工具 B 负责什么 -> 工具 C 负责什么 -> 给组合顺序。

## 6. 提示词模板型

展示坏输出 -> 拆模板字段 -> 填一个真实例子 -> 展示好输出 -> 给完整模板。

## 7. 案例拆解型

展示案例结果 -> 拆结构 -> 拆证据 -> 拆节奏 -> 总结可复用经验。

## 8. 避坑清单型

列出高频坑 -> 每个坑给一个画面证据 -> 给替代做法 -> 结尾形成清单。
